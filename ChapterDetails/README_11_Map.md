**引入Leaflet地图库**

1. 直接使用`script`标签引入

2. `npm install leaflet`

> 总结：
> 该项目主要使用到`leaflet`地图库进行实现，以锻炼对象为主，做的一个面向对象编程的地图项目，主要分为跑步子类和骑车子类，涉及到属性框的设置和消失，同时使用`localstorage`对象，保存这些类对象，以便于在重新打开页面后可以重新显示，主要展示了类的使用。
